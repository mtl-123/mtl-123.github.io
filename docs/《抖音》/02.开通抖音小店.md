# 开通抖音小店

## 准备营业执照

- 办理一个营业执照以便开通抖音小店。

## 注册抖音小店

- 根据官方指引，使用营业执照注册抖音小店，并设置店铺信息。

## 无货源模式入门

- 学习如何利用互联网的信息差，挑选优质商品上传至自己的店铺，不需要自己囤货。
