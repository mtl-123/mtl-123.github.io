# 🧰 使用 `uv` + Fish Shell 的完整配置指南

> 本指南详细介绍了如何在 **Fish Shell** 环境中安装并配置 `uv`，包括添加路径、设置别名、激活虚拟环境等完整流程。适用于 macOS 和 Linux 用户。

---

## 📦 一、什么是 uv？

`uv` 是一个超快的 Python 包管理工具，由 Rust 编写，旨在替代传统的 `pip`, `venv`, `pip-tools` 等工具。它速度快、支持现代 Python 工程化标准（如 `pyproject.toml`），是 Python 开发者的理想选择。

GitHub 地址：[https://github.com/astral-sh/uv](https://github.com/astral-sh/uv)

---

## 🔧 二、安装 uv

### ✅ 方法一：通过官方脚本安装（推荐）

```bash
curl -LsSf https://install.python-poetry.org | sh
```

这会将 `uv` 安装到 `$HOME/.local/bin/uv`。

> ⚠️ 注意：该脚本也会安装 Poetry，如果你只想安装 `uv`，请使用 pip 或手动下载二进制文件。

---

## 🐟 三、Fish Shell 配置

### 1. 检查是否已加入 PATH

```fish
which uv
```

- 如果输出路径如 `/Users/xxx/.local/bin/uv`，说明已加入。
- 如果没有输出，请继续下面步骤。

---

### 2. 手动将 `uv` 添加到 PATH（Fish Shell）

编辑 Fish 配置文件：

```bash
nano ~/.config/fish/config.fish
```

添加如下内容（根据你的实际路径调整）：

```fish
set -gx PATH $PATH "$HOME/.local/bin"
```

保存后重新加载配置：

```fish
source ~/.config/fish/config.fish
```

验证是否成功：

```fish
uv --version
```

你应该看到类似输出：

```
uv 0.5.0 (xxxxx)
```

---

## ⚙️ 四、Fish Shell 中的快捷配置（可选但推荐）

为了提升开发效率，我们可以在 Fish Shell 中添加一些常用命令的别名和函数。

### 1. 添加常用别名

在 `~/.config/fish/config.fish` 中添加以下内容：

```fish
alias uvi 'uv install'
alias uvr 'uv remove'
alias uvu 'uv update'
alias uvs 'uv sync'
alias uvpi 'uv pip install -r requirements.txt'
alias uvfr 'uv pip freeze > requirements.txt'
```

### 2. 创建快速激活虚拟环境的函数

```fish
function activate
    if test -f .venv/bin/activate.fish
        source .venv/bin/activate.fish
    else
        echo "No .venv found."
    end
end
```

这样你就可以直接运行：

```fish
activate
```

来激活当前目录下的 `.venv` 虚拟环境。

---

## 🌱 五、创建并使用虚拟环境

### 1. 创建虚拟环境

```fish
uv venv
```

默认会在当前目录下生成 `.venv` 目录。

### 2. 激活虚拟环境（Fish Shell）

```fish
source .venv/bin/activate.fish
```

你可以用前面定义的 `activate` 函数简化操作。

---

## 📂 六、项目依赖管理示例

### 1. 创建一个测试项目

```bash
mkdir myproject && cd myproject
uv venv
```

激活虚拟环境：

```fish
activate
```

---

### 2. 安装依赖包

```fish
uvi requests pandas
```

或者使用别名：

```fish
uv install requests numpy
```

---

### 3. 导出依赖

```fish
uvfr
cat requirements.txt
```

你会看到类似：

```
requests==2.31.0
pandas==2.2.2
numpy==2.0.2
```

---

### 4. 从依赖安装

```bash
rm -rf .venv
uv venv
source .venv/bin/activate.fish
uvpi
```

这会重建虚拟环境并安装所有依赖。

---

## 🧪 七、配合 pyproject.toml 使用（现代 Python 项目）

如果你有一个包含 `pyproject.toml` 的项目，可以直接使用：

```bash
uv sync
```

这会根据 `[project.dependencies]` 安装所有依赖，并生成或更新 `uv.lock` 文件。
