以下是根据提供的资料整理的 UFW (Uncomplicated Firewall) 命令详解，以 Markdown 格式呈现：

# UFW (Uncomplicated Firewall) 命令详解

# 简介

UFW是一款基于iptables构建的、用于管理防火墙规则的用户友好型工具。它简化了在Linux系统上配置防火墙的过程。

# 安装

# 在Ubuntu/Debian上安装

    sudo apt update
sudo apt install ufw

    // Make sure to add code blocks to your code group

    # 示例用法

    # 查看当前ufw的状态

    sudo ufw status
    1

    # 查看ufw的详细状态信息

    sudo ufw status verbose
    1

    # 启用ufw

    sudo ufw enable
    1

    # 禁用ufw

    sudo ufw disable
    1

    # 开启日志

    sudo ufw logging on
    1
    2

    # 拒绝所有传入连接

    sudo ufw default deny incoming
    1

    # 允许所有传出连接

    sudo ufw default allow outgoing
    1

    # 允许指定的流量

    sudo ufw allow <port/service>
    1

    # 允许HTTP 80端口

    sudo ufw allow 80
    1

    # 允许HTTPS 443端口

    sudo ufw allow 443
    1

    # 允许SSH服务

    sudo ufw allow ssh
    1

    # 拒绝流量

    sudo ufw deny <port/service>
    1

    # 拒绝FTP 21端口

    sudo ufw deny 21
    1

    # 允许来自特定IP的流量

    sudo ufw allow from <IP>
# 例如：
sudo ufw allow from 192.168.1.100
    1
    2
    3

    # 允许从特定IP到特定端口的流量

    sudo ufw allow from <IP> to any port <port>
# 例如：
sudo ufw allow from 192.168.1.100 to any port 22
    1
    2
    3

    # 允许子网的流量

    sudo ufw allow from <subnet>
# 例如：
sudo ufw allow from 192.168.1.0/24
    1
    2
    3

    # 移除指定的规则

    sudo ufw delete allow <port/service>
# 例如：
sudo ufw delete allow 80
    1
    2
    3

    # 按编号删除规则

    先用sudo ufw status numbered查看编号 再执行：

    sudo ufw delete <rule_number>
# 例如：
sudo ufw delete 3
    1
    2
    3

    # 启用ufw日志

    日志记录在/var/log/ufw.log文件

    sudo ufw logging on
    1

    # 设置日志级别

    可用的日志级别有：low, medium, high, full

    sudo ufw logging medium
    1

    # 禁用ufw日志

    sudo ufw logging off
    1

    # 限制ssh连接尝试次数

    sudo ufw limit ssh
    1

    # 向发送者发送拒绝响应

    sudo ufw reject <port/service>
# 例如：
sudo ufw reject 8081
    1
    2
    3

    # 启用IPv6支持

    编辑ufw配置文件：/etc/ufw/ufw.conf，添加如下行：

    IPV6=yes
    1

    # 重置ufw为默认设置

    将禁用ufw，删除所有规则并重置为默认配置。

    sudo ufw reset
    1

    # 列出可用的应用程序配置文件

    sudo ufw app list
    1

    # 允许一个应用的流量

    sudo ufw allow <app_name>
# 例如：
sudo ufw allow nginx
    1
    2
    3

    # 显示应用流量配置详情

    sudo ufw app info <app_name>
# 例如：
sudo ufw app info nginx
    1
    2
    3

    # 允许MySql 3306端口

    sudo ufw allow from 192.168.1.0/24 to any port 3306
    1

    # 允许范围端口

    sudo ufw allow 6000:6007/tcp
sudo ufw allow 6000:6007/udp
    1
    2

    # 重载修改的配置

    sudo ufw reload
    1
