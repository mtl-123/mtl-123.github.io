uv&Python

uv 安装 (opens new window)

以下是一份使用 uv 搭配 Fish Shell 的完整指南（含安装、配置与验证全过程），教程使用。

# 🧰 使用 uv + Fish Shell 的完整配置指南

本指南详细介绍了如何在 Fish Shell 环境中安装并配置 uv，包括添加路径、设置别名、激活虚拟环境等完整流程。适用于 macOS 和 Linux 用户。

# 📦 一、什么是 uv？

uv 是一个超快的 Python 包管理工具，由 Rust 编写，旨在替代传统的 pip, venv, pip-tools 等工具。它速度快、支持现代 Python 工程化标准（如 pyproject.toml），是 Python 开发者的理想选择。

GitHub 地址：https://github.com/astral-sh/uv (opens new window)

# 🔧 二、安装 uv

# ✅ 方法一：通过官方脚本安装（推荐）

curl -LsSf https://install.python-poetry.org | sh
1

这会将 uv 安装到 $HOME/.local/bin/uv。

⚠️ 注意：该脚本也会安装 Poetry，如果你只想安装 uv，请使用 pip 或手动下载二进制文件。

# 🐟 三、Fish Shell 配置

# 1. 检查是否已加入 PATH

which uv
1
  • 如果输出路径如 /Users/xxx/.local/bin/uv，说明已加入。
  • 如果没有输出，请继续下面步骤。

# 2. 手动将 uv 添加到 PATH（Fish Shell）

编辑 Fish 配置文件：

nano ~/.config/fish/config.fish
1

添加如下内容（根据你的实际路径调整）：

set -gx PATH $PATH "$HOME/.local/bin"
1

保存后重新加载配置：

source ~/.config/fish/config.fish
1

验证是否成功：

uv --version
1

你应该看到类似输出：

uv 0.5.0 (xxxxx)
1

# ⚙️ 四、Fish Shell 中的快捷配置（可选但推荐）

为了提升开发效率，我们可以在 Fish Shell 中添加一些常用命令的别名和函数。

# 1. 添加常用别名

在 ~/.config/fish/config.fish 中添加以下内容：

alias uvi 'uv install'
alias uvr 'uv remove'
alias uvu 'uv update'
alias uvs 'uv sync'
alias uvpi 'uv pip install -r requirements.txt'
alias uvfr 'uv pip freeze > requirements.txt'

# 2. 创建快速激活虚拟环境的函数

function activate
    if test -f .venv/bin/activate.fish
        source .venv/bin/activate.fish
    else
        echo "No .venv found."
    end

# 🌱 五、创建并使用虚拟环境

# 1. 创建虚拟环境

uv venv
1

默认会在当前目录下生成 .venv 目录。

# 2. 激活虚拟环境（Fish Shell）

source .venv/bin/activate.fish
1

你可以用前面定义的 activate 函数简化操作。

# 📂 六、项目依赖管理示例

# 1. 创建一个测试项目

mkdir myproject && cd myproject
uv venv
1
2

激活虚拟环境：

activate
1

# 2. 安装依赖包

uvi requests pandas
1

或者使用别名：

uv install requests numpy
1

# 3. 导出依赖

uvfr
cat requirements.txt
1
2

你会看到类似：

requests==2.31.0
pandas==2.2.2
numpy==2.0.2
1
2
3

# 4. 从依赖安装

rm -rf .venv
uv venv
source .venv/bin/activate.fish
uvpi
1
2
3
4

这会重建虚拟环境并安装所有依赖。

# 🧪 七、配合 pyproject.toml 使用（现代 Python 项目）

如果你有一个包含 pyproject.toml 的项目，可以直接使用：

uv sync
1

这会根据 [project.dependencies] 安装所有依赖，并生成或更新 uv.lock 文件。

# ✅ 八、完整验证流程总结

步骤 命令
安装 uv curl -LsSf https://install.python-poetry.org | sh
加入 PATH set -gx PATH $PATH "$HOME/.local/bin"
验证版本 uv --version
创建虚拟环境 uv venv
激活虚拟环境 source .venv/bin/activate.fish
安装依赖 uv install requests 或 uv pip install -r requirements.txt
卸载依赖 uv remove requests
更新依赖 uv update requests
同步依赖 uv sync
导出依赖 uv pip freeze > requirements.txt
